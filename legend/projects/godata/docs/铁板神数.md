**一、概念总览**

| 关键要素      | 含义与作用                                                                                                                                                      |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **“数”体系** | 以 **天干地支→数字**（后天八卦数 1–9）为核心：<br>• **天数** = 出生年 + 月 + 日 + 时换算之数（代表“天命”）<br>• **人数** = 占算当下年 + 月 + 日 + 时换算之数（代表“人事”）<br>两者相合（或差）再融入**地数**（洛书河图等）推衍六亲、宫位与条文索引 |
| **十二程序**  | 铁板神数把 12 000 条文映射到 12 步推演流程；每完成一步就锁定一组条文并检核出生时刻的准确性，最终汇编成完整命书 ([Wikipedia][1])                                                                             |
| **基础工具**  | 四柱八字、64 卦象、《太玄经》数理、河洛理数、纳音五行、六亲（父母/兄弟/夫妻/子息/财帛/福德）体系                                                                                                       |
| **精度要求**  | 重视“定刻”——时间需细到 **1 刻 = 15 分钟**，这是其自诩精准的根源 ([Wikipedia][1])                                                                                                  |

**二、完整推演流程（实操视角）**

> ⚠️ 以下为传统术式总结，缺乏科学证据，仅供了解文化脉络。

1. **收集出生信息**

   * 阳历/阴历均可，先统一历法。
   * 精确到出生时辰并细分 15 分钟刻度；若无法确定，需先用人生关键事件“校时”。

2. **换算四柱→数字（起 天数）**

   * 依后天八卦数：乾 6、兑 7、离 9、震 3、巽 4、坎 1、艮 8、坤 2。
   * 例：甲（属乾）取 6，乙（坤）取 2……以此转换年干、支，月、日、时共 8 字，合并为天数。

3. **占算时刻换算（起 人数）**

   * 用同一套规则把“问命当下”的年/月/日/刻化数，得人数。
   * 人数提供“人事场”变量，兼具校时功能。若推断与现实不符，多半是出生刻度偏差，需回到步骤 1 修正。

4. **合天人→定六亲宫**

   * 将天数、人数进行合/差/倍/除等固定算法，依次推得：命宫、父母宫、兄弟宫、夫妻宫、子息宫、财帛宫……
   * 每个宫位再映射到 I Ching 64 卦（例：数 37→卦 风火家人）。

5. **套入河洛/太玄数交叉验证**

   * 对应卦象再以河洛理数、太玄数做奇偶、阴阳、五行生克检验，剔除不合理条文索引，确保“条文唯一”。

6. **检索 12 000 条文**

   * 传统以《皇极分经数》或《神机妙算铁板数》版本为基准。
   * 每完成一步就锁定若干行条文；12 步结束后，通常会汇得 50–80 条，覆盖一生重要节点（出身、科名、婚姻、子息、灾厄、寿终等）。

7. **条文整合与断验**

   * 按年表、运限（元/会/运/世）与流年大运顺序编排。
   * 通过对照已发生事件再次验时刻；若误差仍在，可小幅调整刻度重新跑步骤 2–6。
   * 最终形成命书，“铁板”意指条文一经确定便不可更改。

> ⑪迁移 ⑫仆役（福德）
> *每完成一程序即对应一段条文；亦有岭南派将“福德”与“相貌”拆分为 13 步。*

**四、条文使用要点**

* **多义校正**：同一条文可有多层含义，需结合六亲宫位、五行旺衰判定主应在本人还是亲属。
* **流年触发**：条文中的“某年”多以 **纳音五行 + 太岁干支** 指向具体年份，再对照岁运大运。
* **应期前后**：重大吉凶应期常提前或滞后 1 年，校时误差越小偏差越低。

| 场景       | 建议                                                                                           |
| -------- | -------------------------------------------------------------------------------------------- |
| **入门自学** | 先熟练八字排盘 → 学《易经》64 卦象 → 理解后天八卦数与河洛数对应，再读《皇极分经数》条文范例                                           |
| **进阶研习** | 研读少数公开的古籍影印本（如《神机妙算铁板数》道光本），对照现代命例软件。                                                        |
| **实务占算** | • 使用现代排盘程序节省换算时间，但保留手工校时环节<br>• 至少记录并验证 30 个完整命例，累积条文诠释经验<br>• 出断应给出事件区间（××岁前后）而非绝对某日，保留检验弹性 |
| **伦理提醒** | 凡占算结果牵涉病灾、寿限等，宜遵守“适度披露”原则，避免造成心理负担或商业误导。                                                     |
